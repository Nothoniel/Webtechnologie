The browser, Internet Explorer

Written by: Nathan van de Kolk, Joran van de Kolk, Do Toan Nguyen

History

Introduction

Google Chrome is a web browser developed by Google. It was first publicly released on Microsoft Windows on 2008 and later ported to many other platforms [1]. According to StatCounter, as of January 2021, Google Chrome controls 66.59% of the global web browser market and has been the most prominent web browser since 2012 [2].

Announcement

The Browser was planned to be first announced on September 3, 2008 in the form of a 38-page comic book which was sent to various jounalists and bloggers. This comic book would explain all the features of the web browser [3]. It didn't go quite as planned though, since the German blogger Philipp Lenssen published a scan of the comic book on his website on september 1, the same day he received it [4]. The project code name "Chrome" was kept as the product name at release. This name is considered quite ironic though: chrome is often associated with cars and speed, but the browser was designed to be simplistic in contrast to so-called user interface chrome [5].

First Release

Google Chrome was first released on September 2, 2008 for Windows XP as a beta version[1]. Later that year on December 11, the first stable version was released. Alongside the web browser, Google also published the Chromium source code, which the browser was and still is mostly based on [1]. This open source project has made quite the impact on future web browser development since many other browsers, for example the new version of Microsoft Edge, also known as Chromium edge, have used parts of this source code [6].

Chrome OS

Chrome OS is an operating system designed by Google that is based on Gentoo Linux. The project was first announced on July 7, 2008 as an OS with the Chrome browser as its primary component and with a similar UI design as Chrome [7]. The operating system would be one in which all applications and user data would be stored in a cloud instead of a hard drive. The only application installed would therefore be a web browser with other applications like a file explorer and mediaplayer incorporated in it. Laptops running this operating system are also known as Chrome Books. This OS is aimed at users who would use their device almost solely to browse the web, meaning that the main focus lies on speed above all else [8].. Google also requested its partnered hardware manufacturers to use solid-state drives instead of hard disk drives, in order to further increase the speed of the Chrome Books [9].

Other Platforms

The latest verion of Google Chrome is currently available for the following platforms:

  • Microsoft Windows 7 or later;
  • MacOS 10.11 or later;
  • Android 5.0 and later;
  • iOS 12.0 and later;
  • Linux desktop;

Older versions of these platforms may also have an old version of Google Chrome available, but those versions are no longer supported [10].

Features

Incognito Mode

The user can choose to open tabs in Incognito Mode. This feature is quite similar to private browsing modes of other web browsers. When in incognito mode, the browser is unable to permanently store things like cookies, search history, and form inputs. This is only the case for incognito tabs though, normal tabs the user has opened aside from incognito tabs still store information like normal. As mentioned when opening a incognito tab though, the user may not be in complete privacy since the user's visited websites, search engines, organisation or internet service provider may still be able to see their activity [11].

Password Management

On various platforms the user can make use of password management. After logging in on a website for example, the user may be asked whether or not they want their login credentials to be saved.
On Windows, the user will be asked to enter the master password of the Operating system before accessing the password manager [12].
On Linux, passwords can be stored in three different ways:

  • plain text
  • GNOME Keyring
  • KWallet

In all 3 the cases the passwords will be stored somewhere on the disk. With plain text as the setting, the passwords are stored unencrypted, while the other two options do encrypt the data. When switching between plain text and an encrypting option, the data is automatically encrypted. The web browser chooses one of the options automatically, based on the current desktop environment, though the setting can be forced to one of the options by using a command-line [13].

Dinosaur game

When a user tries to browse without internet connection, a message with a picture of an 8-bit T-Rex shows up. When the user then presses either the space bar or the up-arrow key or by either clicking or tapping (on mobile) a game launches where the user controls the dinosaur running through a desert. The player can jump with space, up-arrow or a tap on the screen(on mobile). On desktop the dinosaur can also duck when the down-arrow key is pressed. This move set allows the player to dodge obstacles, which are cacti and pterodactyls. The player get a score depending on how long they survive. The game has no particular end, but it does get faster the further the user goes [14]. This easter egg can be blocked on chrome books by an administrator, in order to prevent the user to be distracted from work [15].
