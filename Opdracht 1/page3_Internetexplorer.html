History

What is it exactly?

First of all what is Internet Explorer exactly? It is a World Wide Web Browser that comes bundled with the Microsoft Windows operating system (OS).
It is one of the mainstream webbrowsers.
IE was the most popular browser from 1999, when it overtook Netscape Navigator, until 2012, when Chrome took the lead [1].

First Version

The first version of Internet Explorer (fig. 1) was started in the summer of 1994 by Thomas Reardon, who,
according to the Massachusetts Institute of Technology Review of 2003 [3], used source code from Spyglass,
Inc. Mosaic, which was an early commercial web browser with formal ties to the pioneering National Center for Supercomputing Applications (NCSA) Mosaic browser [4].

Death of Internet Explorer

Microsoft Edge was released in 2015. The goal of Edge was that it is going to replace Internet Explorer.
But Internet Explorer is still installed in newer versions of Windows , since some older websites do not work with Edge only.
Edge is like the better version of IE, since it can use a lot of extensions like Google chrome and works a lot faster too [5][6].

Features

Most recent version

So for the features only about Microsoft Edge the most recent features are going to be discussed in this section,
since the features of Internet explorer are a bit irrelevant now.
Also disucssing multiple features of countless of iterations of Internet Explorer + Edge is a lot of info [7].

Collections

Collections allows you to easily collect information from different websites, organize it,
export it into various files or just come back to your research at a later time and pick up right where you left off [7].

Vertical Tabs

Today we announced vertical tabs, a feature that helps you easily find and manage many open tabs at once.
Microsoft Edge is the only browser that allows you to manage your tabs on the side with a single click [7].

Smart copy

Smart copy makes it easier to select, copy and paste the exact content you want from the web and preserve the rich web format.
You can use your cursor to select any area or type of content.
When you paste it will retain the formatting, including any images or links [7].

Tracking prevention

The tracking prevention feature in Microsoft Edge is designed to protect you from being tracked by websites that you aren’t accessing directly,
giving you more control over what you see and what you don’t. Whether on mobile or desktop, you can select one of three settings that’s right for you – Basic, Balanced or Strict.
Depending on which setting you choose, Microsoft Edge will adjust the types of third-party trackers blocked [7].

Password Monitor & Private mode

When enabled, Password Monitor is a feature that notifies you if the credentials you’ve saved to autofill have been detected on the dark web.
Private mode is having the option to use the browser while it automatically deletes the history, cookies and site data [7].
